# Mohamed Ezzat Portfolio

Static portfolio website for Mohamed Ezzat, built as a host-ready frontend with no build step required.

## Files

- `index.html` - main site markup
- `styles.css` - visual design and responsive layout
- `script.js` - live GitHub repository loading
- `Mohamed_Ezzat_Resume.pdf` - resume download for visitors
- `.github/workflows/deploy-pages.yml` - automatic GitHub Pages deployment
