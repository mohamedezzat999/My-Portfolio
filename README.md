# Mohamed Ezzat Portfolio

Static portfolio website for Mohamed Ezzat, built as a host-ready frontend with no build step required.

## Files

- `index.html` - main site markup
- `styles.css` - visual design and responsive layout
- `script.js` - live GitHub repository loading
- `Mohamed_Ezzat_Resume.pdf` - resume download for visitors
- `.github/workflows/deploy-pages.yml` - automatic GitHub Pages deployment

## Best Free Hosting

This project is prepared for GitHub Pages.

## Deploy on GitHub Pages

1. Push this folder to a GitHub repository.
2. In GitHub, open `Settings` -> `Pages`.
3. Under `Build and deployment`, choose `GitHub Actions`.
4. Push to `main` or `master`.
5. GitHub will deploy the site automatically using the included workflow.

## Local preview

Open `index.html` directly in a browser, or serve the folder with any static server.
